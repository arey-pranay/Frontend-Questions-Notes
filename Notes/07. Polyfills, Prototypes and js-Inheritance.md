# 1️⃣ What is a Polyfill?

## ✅ Definition


# 3️⃣ `__proto__` vs `prototype`

VERY IMPORTANT confusion.

---

# `prototype`

Exists on:

* constructor functions/classes

Used for inheritance.

---

# Example

```js id="i18"
function User() {}
```

```js id="i19"
User.prototype.sayHi = function() {};
```

---

# `__proto__`

Actual internal prototype link of an object.

---

# Example

```js id="i20"
const obj = {};
```

```js id="i21"
obj.__proto__ === Object.prototype
// true
```

---

# 🚨 Golden Rule

| Thing       | Meaning                 |
| ----------- | ----------------------- |
| `prototype` | property on constructor |
| `__proto__` | link on object instance |

# 5️⃣ Important Polyfills

---

# ✅ `map` Polyfill

```js id="i24"
Array.prototype.myMap = function(callback) {
  const result = [];

  for (let i = 0; i < this.length; i++) {
    result.push(callback(this[i], i, this));
  }

  return result;
};
```

---

# Tests:

* iteration
* callbacks
* `this`

---

# ✅ `filter` Polyfill

```js id="i25"
Array.prototype.myFilter = function(callback) {
  const result = [];

  for (let i = 0; i < this.length; i++) {
    if (callback(this[i], i, this)) {
      result.push(this[i]);
    }
  }

  return result;
};
```

---

# ✅ `reduce` Polyfill

VERY IMPORTANT.

```js id="i26"
Array.prototype.myReduce = function(callback, initialValue) {
  let accumulator = initialValue;
  let start = 0;

  if (accumulator === undefined) {
    accumulator = this[0];
    start = 1;
  }

  for (let i = start; i < this.length; i++) {
    accumulator = callback(accumulator, this[i], i, this);
  }

  return accumulator;
};
```

---

# 🚨 Important Reduce Edge Case

If:

* no initial value
* empty array

Must throw error.

---

# ✅ `forEach` Polyfill

```js id="i27"
Array.prototype.myForEach = function(callback) {
  for (let i = 0; i < this.length; i++) {
    callback(this[i], i, this);
  }
};
```

---

# ✅ `find` Polyfill

```js id="i28"
Array.prototype.myFind = function(callback) {
  for (let i = 0; i < this.length; i++) {
    if (callback(this[i], i, this)) {
      return this[i];
    }
  }
};
```

# ✅ `bind` Polyfill

VERY IMPORTANT.

```js id="i29"
Function.prototype.myBind = function(context, ...args) {
  const fn = this;

  return function(...newArgs) {
    return fn.apply(context, [...args, ...newArgs]);
  };
};
```

---

# Tests:

* `this`
* closures
* apply/call

# ✅ `call` Polyfill

---

# Internal Idea

Temporarily attach function to object.

```js id="i30"
Function.prototype.myCall = function(context, ...args) {
  context.fn = this;

  const result = context.fn(...args);

  delete context.fn;

  return result;
};
```

---

# 🚨 Why It Works

Because:

```js id="i31"
obj.fn()
```

sets:

* `this = obj`

# 6️⃣ `new` Keyword Internals

VERY IMPORTANT.

---

# `new User()`

Internally does:

```text id="i32"
1. creates empty object
2. links prototype
3. binds this
4. executes constructor
5. returns object
```

---

# Simplified Polyfill

```js id="i33"
function myNew(Constructor, ...args) {
  const obj = Object.create(Constructor.prototype);

  const result = Constructor.apply(obj, args);

  return typeof result === "object" ? result : obj;
}
```

---

# 7️⃣ `instanceof` Internals

Checks prototype chain.

---

# Example

```js id="i34"
arr instanceof Array
```

Searches:

```text id="i35"
Array.prototype
```

inside prototype chain.

---

# Simplified Polyfill

```js id="i36"
function myInstanceOf(obj, Constructor) {
  let proto = Object.getPrototypeOf(obj);

  while (proto) {
    if (proto === Constructor.prototype) {
      return true;
    }

    proto = Object.getPrototypeOf(proto);
  }

  return false;
}
```

---

# 8️⃣ Object.create Polyfill

```js id="i37"
function myCreate(proto) {
  function F() {}

  F.prototype = proto;

  return new F();
}
```
